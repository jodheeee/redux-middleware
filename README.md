## redux middleware form

## 디렉토리 구조
```
  (src)
    │
    ├─ api : API 비동기 통신 요청 작성
    │
    ├─ components : 프리젠테이셔널 컴포넌트로 view 단(HTML Tag 작성 등)
    │
    ├─ containers : redux에 있는 state를 조회하거나 action을 dispatch 할 수 있는 컴포넌트
    │
    ├─ lib : 반복되는 소스를 리팩토링 하기 위한 유틸을 작성
    │
    ├─ modules : redux의 state, reducer를 정의한 파일
    │
    └─ pages : 페이지
    
```
